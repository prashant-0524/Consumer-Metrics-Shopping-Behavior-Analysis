# Retail Consumer Shopping Behavior Analysis 📊🛍️

## 📌 Project Overview
This repository contains an end-to-end data analytics project focused on understanding customer shopping behavior for a leading retail company. The core objective is to leverage consumer data to identify trends, improve customer engagement, and optimize marketing and product strategies to drive long-term loyalty.

The project seamlessly integrates **Python** for data cleaning, **SQL** for deeper transactional querying and customer segmentation, and **Power BI** for building an interactive, executive-ready dashboard.

---

## 🛠️ Tech Stack & Tools
* **Data Preparation & Engineering:** Python (Pandas, NumPy)
* **Database & Analytical Querying:** SQL (PostgreSQL / MySQL)
* **Data Visualization & BI:** Power BI
* **Version Control:** GitHub
