# Retail Consumer Shopping Behavior Analysis 📊🛍️

## 📌 Project Overview
This repository contains an end-to-end data analytics project focused on understanding customer shopping behavior for a leading retail company. The core objective is to leverage consumer data to identify trends, improve customer engagement, and optimize marketing and product strategies to drive long-term loyalty.

The project seamlessly integrates **Python** for data cleaning, **SQL** for deeper transactional querying and customer segmentation, and **Power BI** for building an interactive, executive-ready dashboard.

---

## 🛠️ Tech Stack & Tools
* **Data Preparation & Engineering:** Python (Pandas, NumPy)
* **Database & Analytical Querying:** SQL (PostgreSQL / MySQL)
* **Data Visualization & BI:** Power BI
* **Version Control:** GitHub

💡 Key Insights & Strategies
The Subscription Lever: 73% of the customer base operates transactionally (Non-Subscribers). Converting a fraction of these via loyalty incentives is a massive opportunity for repeat revenue.

Product Bundling: Clothing (104.26K) and Accessories (74.20K) drive the majority of revenue. We can lift underperforming segments like Footwear by introducing "Complete the Look" bundle discounts at checkout.

Seasonal Slumps: Revenue consistently dips from Fall (60.0K) down to Summer (55.8K). Targeted end-of-season clearance campaigns can help stabilize this trajectory.

Customer Satisfaction: The average review rating is 3.75/5, signaling room for operational or product quality improvements, while highly distributed payment preferences show a friction-free checkout process.
